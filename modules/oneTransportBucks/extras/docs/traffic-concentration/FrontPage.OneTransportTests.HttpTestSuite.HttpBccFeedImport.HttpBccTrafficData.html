As an Application I want to access traffic data

As an Application I want to access traffic data

Traffic data is stored under two Application Entities (AE): BCCTrafficFlowFeedImport and BCCTrafficFlowExtensionFeedImport. Each AE corresponds to an DATEX2 feed provided by the Local Authority.

In the CSE hierarchy directly beneath each AE, on the second-level, are data containers named after a location reference. For the BCCTrafficFlowFeedImport AE on the 3rd level are data containers for trafficFlow, trafficSpeed and travelTimeValue. For the BCCTrafficFlowExtensionFeedImport there are data containers for trafficQueue and trafficScoot. Each of the containers at the third level of the hierarchy contain JSON objects that have DATEX2 attributes.

Beside the containers for locations, on the second-level there is also a container All whose content instances will contain an array of all of the latest JSON objects created.

The following diagram shows the hierarchy of data containers and the JSON objects which are placed within them:
- BCCTrafficFlowFeedImport
  - LINKBUCK-110NW1
      - trafficFlow        
        - la = { "locationReference": "LINKBUCK-110NW1", "vehicleFlow": ? }
      - trafficSpeed 
        - la = { "locationReference": "LINKBUCK-110NW1", "averageVehicleSpeed": ? }
      - trafficConcentration
      	- la = { "locationReference": "LINKBUCK-110NW1", "occupancy": ? }
  - Section1001
      - travelTimeValue
        - la = {"locationReference": "Section1001", "travelTime": ?, "freeFlowSpeed": ?, "freeFlowTravelTime": ?, "averageVehicleSpeed": ?, "vehicleFlow": ?}
  - All
- BCCTrafficFlowExtensionFeedImport
  - LINKBUCK-110NW1
      - trafficQueue
        - la = { "locationReference":"LINKBUCK-110NW1", "queuePresent": ?, "queueSeverity": ? }
      - trafficScoot
        - la = { "locationReference": "LINKBUCK-110NW1", "congestionPercent": ?, "currentFlow": ?, "averageSpeed": ?, "linkStatus": ?, "linkStatusType": ?, "linkTravelTime": ?

The LINKBUCK-110NW1 container is named after a link section. This can be used to correlate with the location information.
